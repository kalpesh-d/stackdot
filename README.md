## Getting Started

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/kalpesh-d/stackdot.git
   cd stackdot
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

### Running the App

```sh
npm run dev
```
